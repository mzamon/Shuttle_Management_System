## Project Overview

This repository contains the Shuttle Booking Management System developed for NBK Travel, a small shuttle transport company. The system replaces manual booking processes — telephone reservations, paper registries, and personal spreadsheets — with a centralised digital platform featuring automated scheduling, customer record management, and business intelligence reporting.

This project is developed as part of the academic Work Integrated Learning (WIL) module requirements for the Diploma in IT in Software Development.

## Key Features

- **Booking Management Module** — Digital capture, storage, and retrieval of all trip bookings with real-time availability checking
- **Scheduling System** — Automated driver and vehicle assignment with conflict detection
- **Customer Database** — Centralised records for new, repeat, and potential clients
- **Reporting Module** — On-demand reports for trips, revenue, driver utilisation, and route performance
- **Invoice Generator** — Automated PDF invoice creation
- **Notification Simulation** — Booking confirmations and reminders
